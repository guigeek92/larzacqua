# Démarrage Rapide - SimuWatter

## ⚡ En 3 étapes

### 1️⃣ Installation (première fois seulement)

**Windows :**
```bash
setup.bat
```

**Linux/Mac :**
```bash
./setup.sh
```

### 2️⃣ Configuration

Éditer le fichier `.env` et ajouter votre clé API Groq :
```
GROQ_API_KEY=votre_clé_ici
```

[Obtenir votre clé API Groq](https://console.groq.com)

### 3️⃣ Lancer l'application

**Windows :**
```bash
run.bat
```

**Linux/Mac :**
```bash
./run.sh
```

L'interface ouvrira automatiquement sur : **http://localhost:8501**
